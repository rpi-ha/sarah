Use the following to maintain S.A.R.A.H. - (be sure to generate and install
the config files, using the Home Wizard, after you make changes)

 
General settings
Morning time (e.g. 06:00 for 6:00am) in 24 hour time (add 12 after noon)
Daytime (e.g. 10:00 for 10:00am) in 24 hour time (add 12 after noon)
Evening time (e.g. 17:00 for 5:00pm) in 24 hour time (add 12 after noon)
Night time (e.g. 23:00 for 11:00pm) in 24 hour time (add 12 after noon)
 
Home description (e.g. Seattle, WA)
Home location latitude (e.g. 47.608013)
Home location longitude (e.g. -122.335167)
Home location altitude (e.g. 45) in Meters (ft / 3.2808)
Home location refresh rate (e.g. 180 secs)
find your coordinates - https://www.latlong.net/
 

 
Email settings
Email to receive messages separate with ';'
         (e.g. <your email>@gmail.com)
Hostname - The email server hostname (e.g. smtp.gmail.com)
Email port - defaults to 25 (e.g. 587 for TLS/SSL, 465 for SSL)
Username - if the email server requires authentication
Password - if the email server requires authentication
Email address to use for sending mails from (e.g.
         <your email>@gmail.com)
Use STARTTLS - check this if enabled on mail server (defaults to
         false)
Use SSL negotiation - check this if enabled on mail server (defaults
         to false, do not use both starttls=true and ssl=true)
POP before SMTP - check this if you use POP instead of SMTP
         (defaults to false)
Character set used to encode message body (platform default is
         used if blank)
find some common email configurations - https://rpi-ha.com/the-alarm-system/#setting-up-email-alerts
 

 
Vacation mode settings

Wake time (e.g. 07:00 for 7:00am) your average wake time, in 15 minute increments - in 24 hour time (add 12 after noon)
Bed time (e.g. 23:00 for 11:00pm) your average bedtime, in 15 minute increments - in 24 hour time (add 12 after noon)
Bedtime is next day check this if you go to bed the day after you wake up (usually for swing shift work)
